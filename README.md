# 🥗 DietBite

**Portfolio Showcase**  
DietBite is an Indian nutrition management platform designed to help dietitians, nutritionists, and health-conscious users create meal plans, calculate nutritional values, and manage patient information using Indian food composition data.

> **Note:** This repository contains project documentation, screenshots, and architecture only. The source code is kept private.

---

## 📌 Project Overview

DietBite aims to simplify nutrition planning by providing tools for:

- Patient Management
- Meal Planning
- Nutrient Calculation
- Food Search
- Secure Authentication
- Role-Based Access Control
- Admin Dashboard
