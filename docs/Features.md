# Features

## Authentication

Users can register and log in securely using JWT authentication.
The application supports role-based access control for Admins, Dietitians, and Users.

---

## Patient Management

Dietitians can:
- Add patients
- Edit patient information
- Track progress
- Manage consultations

---

## Meal Planner

- Create meal plans
- Assign foods
- Calculate nutrients
- Save templates

---

## Food Search

Search Indian foods using the IFCT database with nutritional information.
